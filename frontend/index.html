Motivation

Inclusivity and accessibility are essential in today's interconnected world, yet many communities, such as deaf and hard-of-hearing individuals or non-native speakers, continue to face significant barriers in communication. To address these challenges, our project aims to bridge communication gaps by developing a system that enables real-time, bidirectional translation between speech, text, and sign language.


Help Improve the Model

What TranSlaDoorss Does

TranSlaDoorss is a real-time accessibility tool that helps bridge communication gaps by translating between sign language, text, and speech—supporting more inclusive conversations in classrooms, public services, and everyday interactions.

Sign → Text (Webcam)

Using a webcam stream, the system captures hand/gesture frames and predicts the intended sign. The predicted label is displayed instantly, and you can optionally show confidence to understand model certainty.

Human Feedback Loop

After each prediction, users can mark results as correct or provide the right label. This creates a lightweight feedback dataset that can be used to improve model performance and reduce bias over time.
